# The Glass Bead Game

## Set up
Enter the nix shell:
```
nix-shell
```

Install the root folder:
```
npm install
```

Install the client folder (must use yarn):
```
cd client
```

```
yarn install
```

Change directory back to the root folder:
```
cd ..
```

## Start
```
npm start
```

## Start network
Include the number of agents you want to boot up after 'network'
```
npm run network 2
```
